\begin{jabstract}
近年、スマートフォンや天気アプリの普及により、誰もが簡単に天気予報を確認できるようになった。しかし実際には、天候の影響を日常生活の判断に取り入れることは意外と複雑であり、判断ミスや見落としによって不便やストレスを感じる場面も多い。たとえば、傘を持たずに出かけて突然の雨に降られたり、洗濯物を外に干したまま天候が急変したりすることがある。こうした「備えの不十分さ」による影響は、生活の質や気分にも大きく関わってくる。

この課題を解決するため、本研究では、生成AI（ChatGPT）を活用し、「その日の天候状況に応じた行動提案」を提示するWebシステムを開発した。本システムでは、ユーザが入力した生活習慣──たとえば、天候に依存する家事（洗濯・換気・加湿など）、外出予定の時間帯や内容、所持品（傘や帽子など）、住環境設備（エアコンや除湿機の有無）──と、当日の天気予報（気温・湿度・降水確率など）をもとに、生成AIが具体的な提案文を作成する。

提案内容は、「午後から降水確率が高まるため外出時に傘を持参」「夕方に湿度が上昇するため換気は午前中に実施」「冷え込みが予想される夜間に備えてエアコンのタイマーを活用」といった、天候変化と生活スタイルを踏まえた実用的なアドバイスで構成されている。これにより、ユーザは天気の変化を意識することなく、その日の行動を自然に調整でき、雨に濡れるなどの不便やストレスを未然に防ぐことができる。


\end{jabstract}

\chapter{はじめに}
近年、スマートフォンの普及とともに、気象情報へのアクセスは格段に容易になり、誰もが手軽に天気予報を確認できる時代となっている。多くの人々は通勤・通学や家事などの日常的な行動において、天候に応じた判断を下す必要がある。たとえば、洗濯物を外に干すかどうか、傘を持って外出するかどうか、冷暖房をいつ使用するかといった判断は、天気の変化に大きく左右される。

しかし、実際には天候と生活行動の関係は複雑であり、すべての判断を適切に行うことは容易ではない。気温や湿度、降水確率などの要素を踏まえて、自分のスケジュールや生活環境に合わせた最適な行動を毎日考えるのは大きな負担であり、判断ミスや見落としが生じることも多い。その結果、雨に濡れたり、洗濯物が濡れたり、エアコンの使い方を誤って体調を崩すなど、不便やストレスを引き起こす原因となっている。

一方で、近年注目を集めている生成AI（例：ChatGPT）は、文脈に応じた自然な文章の生成や、複雑な条件を踏まえた提案が可能となっており、生活支援ツールとしての応用が期待されている。もし、天気予報と生活習慣の情報をもとに、AIが「今日どのような行動をとると良いか」を自動で提案してくれるシステムがあれば、ユーザは天気を気にすることなく、快適な一日を過ごすことが可能になると考えられる。

本研究では、こうした課題を解決するために、気象情報とユーザの生活習慣に基づいて、当日の行動提案を生成AIが自動生成するWebシステムを開発した。本システムでは、ユーザが登録した生活習慣情報（天候に影響される家事、持ち物・住環境設備・外出予定）と、当日の天気予報（気温・湿度・降水確率・風速・紫外線指数）をもとに、生成AIが文脈に即した提案文を作成・提示する。たとえば、「夕方から雨が予想されるため、傘を持って外出しましょう」「午前中の湿度が低いため、換気は早めに済ませるのが良いでしょう」など、実用的で具体的なアドバイスを得ることができる。

また、本研究ではこのシステムの有用性を検証するために、ユーザシナリオに基づいた評価実験を行った。提案内容の実効性や納得感、利用しやすさなどの観点から客観評価・主観評価を行い、システムの有効性を明らかにした。

本論文の構成は以下のとおりである。第2章では関連研究を紹介し、従来の生活支援システムや気象情報の応用に関する研究を概観する。第3章では本研究で提案するシステムの全体像と構成要素、ならびに具体的な実装方法について述べる。第4章では評価実験の内容とその結果、考察について詳述し、システムの有効性を検証する。最後に第5章では本研究の結論と今後の課題について述べる。

\chapter{提案システム}

\section{全体像}
本システムは，ユーザが事前に登録した生活習慣や住環境の情報と，当日の天気予報データをもとに，生成AI（OpenAI の GPT-4o）を活用して1日の行動提案を柔軟に生成・提示することを目的とする．これにより，ユーザは自ら具体的な行動を考える負担を軽減しつつ，天候に適した家事や持ち物，換気や洗濯のタイミングなどに関するアドバイスを受けることが可能になる．

図 3.1 にシステムの全体像を示す．本システムでは，まずユーザが事前に生活習慣に関する情報（天候に影響される家事、持ち物・住環境設備・外出予定）を入力する．その後，当日の天気予報（気温，湿度，降水確率、風速、紫外線指数）を Open-Meteo API を通じて取得し，これらの情報を組み合わせて生成AIに送信することで，1日の行動提案を自然言語で自動生成する．

このように，ユーザは天候に応じた柔軟な行動アドバイスを得ることができ，たとえば「午後に雨が降るため午前中に洗濯を済ませるべき」といった具体的な指針を得ることで，日々の生活の質向上につなげることができる．また，システムは外出予定の時間帯や住環境設備の有無なども考慮するため，ユーザごとに個別最適化された提案を得ることができる点に特徴がある．
\newpage

\section{動作手順・動作例}
以下に，本システムの主な動作手順を示す．
\begin{enumerate}
\item 
ユーザ入力：ユーザは利用時に，生活に関する基本情報をフォーム形式で入力する．入力項目には，天候に影響される家事（例：洗濯・換気・掃除），所有する設備（例：乾燥機・加湿器・エアコン），持ち物（例：折りたたみ傘・レインコート），外出予定の時間帯（9:00-18:00）などが含まれる．
\item 
天気データ取得：ユーザが提案ボタンを押すと，Open-Meteo API によりあらかじめ指定した場所に基づいた当日の天気予報（時間単位の気温・湿度・降水確率・風速・紫外線指数）を取得する．このデータは外出予定時間帯や家事提案に反映される．
\item 
プロンプト生成とAI呼び出し：取得した天気情報とユーザ入力情報をもとに，あらかじめ定義されたプロンプトテンプレートにデータを埋め込み，OpenAI API（GPT-4o）に送信する．
\item 
提案表示：GPT-4oの出力を受け取り，Webアプリ上に表示する．洗濯や換気など具体的な行動が，「いつ」「なぜ」行うと良いかの理由とともに自然言語で提示される．
\item 
ユーザ確認・再提案：ユーザは提案を確認し，必要に応じて再生成を行うことができる．また，状況に応じて入力内容を変更し，提案の柔軟な再取得が可能である．
\end{enumerate}

図 3.2 に，実際のプロンプトとGPT-4oから得られた提案結果の例を示す．この例では，〜などユーザ環境に最適化された行動指針が生成されている．

図 3.3 に，本システムのユーザインタフェースを示す．最初の画面で家事として洗濯、持ち物として傘・折り畳み傘・日焼け止めを手動で入力、住環境設備としてエアコンがあるにチェックをする、外出スケジュールに外出している時間帯を入力し、「提案を受け取る」ボタンを押すことで，当日の天候に応じたアドバイスを受け取ることができる．
